Overview
This project contains API automation tests developed as part of the QA Automation technical test for ADL Indonesia.

Project Structure
Key folders in this project:

text
.
├── Test Cases                # Individual test scripts
├── Object Repository         # API request definitions and endpoints
├── Test Suites               # Collections of test cases for execution
├── Reports                   # Generated test execution reports
├── Schemas                   # JSON schema files for response validation
└── README.md
How to Run the Tests
Download the ZIP file and extract the contents to a local folder.

Open Katalon Studio, then select Open Project and choose the extracted folder.

Navigate to the desired Test Cases or Test Suites, then click the Run button to execute.

How to View Test Reports
After running a test suite, the execution report is automatically generated and saved in the Reports folder. You can open the HTML report in a browser to review the results.
